# Frontend To-Do Page

Open `index.html` in a browser or serve the `frontend` folder with a static server.

Quick start (from the `frontend` folder):

```bash
# Python 3
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Features:
- Form with Item Name and Item Description
- Client-side validation (name required)
- Items persisted in `localStorage`
- Delete items
